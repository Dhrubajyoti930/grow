42

For decades, mathematicians were haunted by a single, stubborn integer. The problem was simple to state but monstrous to compute: Can 42 be expressed as the sum of three cubes?

The equation x³ + y³ + z³ = k had been solved for every number under 100, except for two: 33 and 42. In 2019, Andrew Booker and Andrew Sutherland finally cracked the code using Charity Engine—a global grid of half a million home computers.

(-80538738812075974)³ +
(80435758145817515)³ +
(12602123297335631)³ = 42

The solution proved that even the most trivial-looking questions can require planetary-scale collaboration to answer.

42 = 42. Verified.
